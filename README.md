# openapi-autogpt

Based on [openai-chatbot-nodejs](https://github.com/jeromeetienne/openai-chatbot-nodejs)


```
npm install
```

## How to use

First start the server

```
npm run start:server
```

Then start the client

```
npm run start:client
```
